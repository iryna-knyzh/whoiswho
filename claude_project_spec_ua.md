# Проєкт: Хто поруч із вами? / Співзалежиметр 3000

## Ідея
Гумористичний вірусний веб-застосунок про стосунки.
Стиль: меми + психологія у жартівливій формі + відчуття "хочеться відкрити і потикати".

Не робити серйозний психологічний сервіс.
Не використовувати формулювання діагнозів.
Тон: смішно, тепло, іронічно, ніби подруга надіслала посилання вночі.

## Технології MVP

- React
- Vite
- TypeScript
- Локальні JSON-дані
- Без backend
- Без AI у першій версії

## Головний сценарій

"Діагностика стосунків з науковістю приблизно як у гороскопа"

"Почати сканування 👀"

## Логіка MVP

1. Показати 20 запитань
2. У кожної відповіді є тип персонажа
3. Порахувати бали
4. Визначити персонажа-переможця
5. Випадково вибрати готовий фінальний результат з масиву

Використовувати 5–10 заздалегідь написаних варіантів для кожного персонажа.

Без AI генерації.
